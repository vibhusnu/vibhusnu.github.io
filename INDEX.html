FIREBASE SETUP REQUIRED

To enable cross-device sync, you need a free Firebase project. Takes 3 minutes:

  1. Go to console.firebase.google.com → New Project
  2. Add Web App → copy the firebaseConfig object
  3. Authentication → Sign-in method → Enable Email/Password
  4. Firestore Database → Create in production mode
  5. In this HTML file, replace the firebaseConfig block (near the top of the <script type="module">) with your copied config
